# GestureSpeak

**Touchless Communication System using Gestures, Air Writing, and AI**

## Live Demo
## gesturespeakk.netlify.app


## Overview
GestureSpeak AI is a web-based communication system that enables users to communicate without speaking, typing, or touching any device.
Users can interact using simple hand gestures or by writing in the air, and the system converts this input into text, enhances it using AI, and outputs it as speech.

## Problem Statement
People with speech or hearing disabilities often face challenges in communication. Existing solutions require learning sign language or typing, which may not always be fast or accessible.


## Solution
GestureSpeak  allows users to:

* Communicate using hand gestures 
* Write words in the air 
* Generate meaningful sentences using AI 
* Convert text into speech 


##  Features

## **Gesture Recognition**
## Predefined gestures for quick communication (Yes, Help, Stop)

##  **Air Writing**
##  Users can write letters in the air using finger tracking

## **Macro Sentences**
##  * Combination of gestures to form meaningful phrases

## **AI-Based Expansion**
## * Converts minimal input into expressive communication (optional)

##  **Text-to-Speech**
##  Converts output text into voice in real-time

##  **Web-Based Application**
## * No installation required, works directly in browser


##  Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Computer Vision:** MediaPipe Hands
* **Speech:** Web Speech API
* **AI:** OpenAI / Claude (optional)
* **Deployment:** Netlify


##  How It Works

1. User gives input via gestures or air writing
2. System detects and converts input into text
3. AI (optional) enhances the text into meaningful sentences
4. Output is displayed and can be converted into speech
